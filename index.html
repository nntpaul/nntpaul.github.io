Lua Base64 Converter

How to use:

1. Select a file from the dropdown or use "--Custom Tweaks--" to input your own code.

2. Click "Compress & Convert" to process the code.

3. Copy the compressed or Base64 output using the copy buttons.

4. The options from dropdown are pull directly from Backbash repo so it is always updated.
